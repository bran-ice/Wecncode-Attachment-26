Here’s the complete feature list you’ll build.

You can add new prompts with a title, a category, a set of tags, and the full prompt text. You’ll store all those fields for each entry.

You can view all saved prompts so you can browse everything in your library.

You can search by keyword to find prompts that match words in the title, tags, or prompt text.

You can filter by category to see only prompts in a specific category.


You can toggle favorites on prompts so you can mark and quickly access your best ones.
You can edit and delete prompts to keep your library organized and up to date.

You can export filtered prompts to a file so you can share or backup a selected set.

The library saves to a file so your prompts persist between sessions — when you close and reopen the app, your saved prompts remain available.
