# East Africa Tech Company Classification Matrix

This document provides a list of 30 technology companies operating within Kenya and the East Africa region, split into Traditional Software and Artificial Intelligence categories.

---

## 1. Traditional Software Companies (15)
These companies build software using standard human-written rules, focus on digital databases, mobile applications, financial transactions, and enterprise portals.

1. Safaricom (Digital Services) - Manages the core software infrastructure powering M-PESA and merchant applications.
2. Cellulant - Builds digital payment platforms and checkout gateways for merchants.
3. Interswitch East Africa - Focuses on transaction switching and digital payment integration software for banks.
4. Craft Silicon - Designs and installs core electronic banking software across Africa.
5. Finserve Africa - Creates digital banking apps and open banking API integration portals.
6. Techno Brain - Provides enterprise IT solutions, custom code development, and e-government portals.
7. Africa's Talking - Provides communication tools like SMS, USSD, and Voice APIs for developers.
8. CoreTec Systems - Specializes in management software tailored for regional Saccos.
9. Compulynx - Builds retail Point of Sale (POS) and digital identity software.
10. DPO Group - Develops cloud-based payment processing software for online businesses and tourism.
11. Attain Enterprise Solutions - Implements custom ERP systems and business automation platforms.
12. Kopo Kopo - Builds payment processing tools allowing small businesses to accept mobile money.
13. Node Africa - Provides managed cloud storage, server hosting, and digital infrastructure.
14. Visionscape IT - A classic development studio building custom websites and customer web applications.
15. PesaPal - Offers online checkout platforms and physical payment terminal software.

---

## 2. Artificial Intelligence Companies (15)
These companies use Machine Learning, computer vision, or natural language processing. Their software "learns" from massive amounts of data instead of relying purely on human instructions.

1. Amini - Uses machine learning and satellite imagery to provide climate and agricultural data.
2. Fastagger - Provides AI-driven data curation and computer vision models for agriculture and logistics.
3. Sama - A global leader based in Nairobi providing data labeling and verification to train AI models.
4. Leta - Employs predictive AI algorithms to optimize supply chain delivery routes for fleets.
5. Apollo Agriculture - Uses machine learning models on satellite and farm data to predict credit risk for farmers.
6. Tulaa - Evaluates smallholder farmer risk profiles using automated machine learning metrics.
7. UTU Technologies - Builds trust and recommendation engines to identify online fraud patterns.
8. Jacaranda Health - Runs an AI SMS platform (PROMPTS) using local-language NLP to answer medical questions.
9. Tambua Health - Uses deep learning models on lung sounds to diagnose medical conditions via smartphones.
10. Pezesha - Leverages predictive data analytics to evaluate credit scores for small businesses.
11. BasiGo - Uses predictive data tracking to manage electric bus fleets and optimize battery lifetimes.
12. Twiga Foods - Utilizes machine learning engines to forecast market demand and automate supply chains.
13. Integrated Dynamic Solutions (IDS) - Builds predictive customer churn models for financial institutions.
14. Afya Rekod - Uses health tracking data patterns to flag health risks and predict clinical trends.
15. AI Kenya - Develops local AI training workshops and builds custom machine learning prototypes.
